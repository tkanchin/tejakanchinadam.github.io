Projects

Calorie Management System
It is a mobile application which helps users to keep track of thier calories consumed/burnt and users can also view the reports. I have coded a native Android application with JAVA, MySQL datawarehouse (hosted on AWS) and PHP web-services with RESTful JSON.
Wikipedia Vandalism Visualization
To visualize the behavior of vandal users and as a pre-processing to data mining, I have coded this visualiation using d3.js to foresee the outliers and to get insights about the data (Wiki edits).
Prefix search
A naive way of searching words in a dictionary is using Hash Tables. However, it is expensive in terms of both space and time. I have coded a prefix tree (trie) to search words in a dictionary with prefix. The operations such as dictionary construction, search and add/delete/edit are optimized to logarithmic time.
Stress Classification
This is my Thesis project. I have performed data analysis, feature engineering on a time-domain dataset consisiting of millions of rows and classified stress with Semi-Supervised Learning (Co-training & Self-training) with Gradient Tree Boosting. We were first in the literature of stress to classify stress in real-world conditions by the use of wearables.
Image Classification
Optimally classifying images is an open research problem. I have performed multi-class image classification on a data-set consiting of 80,000 high quality images using KNN, Template Matching, Bag of Words, MLP, custom-made CNN and InceptionV3 model and showcased the results.
